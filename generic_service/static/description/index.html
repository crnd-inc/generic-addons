Generic Service


This addon allows you to create and manage service catalog.

You can:

  • create custom Services which can be used in other Odoo apps and modules
  • create custom Service Levels which allow to provide different level of service for customers in various Odoo apps
  • set level of service for each partner on the partner's form

Can be used in other Odoo applications with the help of integration addons. Works great with the Generic Request application as part of the Service Desk system.
